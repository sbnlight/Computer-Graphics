# Computer-Graphics

COMP3271 Computer Graphics — The University of Hong Kong, 2025.

Three programming assignments covering OpenGL, GLSL shaders, and ray tracing.

| Folder | Topic |
|--------|-------|
| `Assignment1/` | OpenGL basics — interactive 2D triangle drawing (Part 1) and 3D mesh editor with rotation / translation / extrusion (Part 2) |
| `Assignment2/` | GLSL vertex and fragment shaders — texture sampling, lighting, and shading on a 3D mesh |
| `Assignment3/` | Path-tracing ray tracer — ray-sphere / ray-triangle intersection, hemisphere sampling, diffuse and mirror reflection |

See each assignment's README for build instructions, tasks, and submission requirements.
